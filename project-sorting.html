PI Conveyor – Object Sorting Mechanism

Mini automation model demonstrating PLC-based sorting using Allen-Bradley RSLogix 500 and FactoryTalk View HMI.

What

  • Manual part sorting on the prototype conveyor was error-prone and inconsistent, especially at varying conveyor speeds.
  • Needed a compact, automated working model to demonstrate real-time sorting using sensors and pneumatic actuation.
  • The objective was to design a functional, educational setup showcasing PLC control, sensor integration, and HMI monitoring.

How

  • Designed a mini conveyor system using photoelectric and inductive sensors to detect object type and position.
  • Programmed an Allen-Bradley MicroLogix PLC in RSLogix 500 to handle object detection, timing, and actuation sequences.
  • Controlled a double-acting pneumatic cylinder for rejection using solenoid valves, ensuring synchronized operation with conveyor movement.
  • Developed a FactoryTalk View HMI for mode selection (Auto / Manual), live sensor feedback, and alarm display.
  • Optimized rung logic for minimal scan delay and implemented interlocks to prevent double actuation or missed parts.

Results

  • Successfully achieved automated sorting of multiple objects based on sensor inputs in real-time.
  • Demonstrated reliable PLC–HMI communication with smooth control over sensors and pneumatic components.
  • Reduced PLC scan cycle time by approximately 20 percent, resulting in faster and more responsive operation.
  • Improved system repeatability and reduced sorting time variability, making it ideal for academic demonstrations and practical automation training.
  • Provided a strong foundation for scaling toward larger industrial conveyor automation systems.
