6D Pose Estimation and Material Handling with ABB GoFa Cobot

Vision-guided handling using Intel RealSense, YOLO, ArUco, and Python → ABB RAPID.

Why

  • Traditional pick-and-place relied on fixed positions, limiting multi-part and variable-orientation handling.
  • Required real-time vision feedback to dynamically adjust object pose and robot orientation.
  • Goal: high-precision handling without re-teaching or manual intervention across cycles.

How

  • Integrated YOLO pose + ArUco marker detection using Intel RealSense depth data.
  • Developed 2D/3D trajectory generation with Python → ABB RAPID socket communication.
  • Performed camera-to-robot calibration for precise spatial mapping.
  • Implemented cycle-based control with Rz quaternion interpolation and offset mapping.

Results

  • Achieved sub-centimeter positional accuracy across multiple cycles.
  • Reduced orientation-switching lag by 40% using optimized Rz-quaternion smoothing.
  • Delivered a modular cobot vision cell adaptable to multiple industrial workflows.
