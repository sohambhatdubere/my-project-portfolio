6D Pose Estimation and Material Handling with ABB GoFa Cobot

Vision-guided handling using Intel RealSense, YOLO, and ArUco with Python to ABB RAPID communication.

Why

  • Traditional pick and place relied on fixed points, making it inefficient for multi-part handling or variable orientations.
  • Needed a real time vision guided control system to adapt dynamically to object position and orientation.
  • The goal was to achieve high precision material handling without manual re teaching between cycles.

How

  • Integrated YOLO pose and ArUco marker detection for 6D pose estimation using Intel RealSense depth data.
  • Developed 2D and 3D trajectory routines with Python to ABB RAPID socket communication for synchronized motion.
  • Performed camera to robot frame calibration to maintain spatial alignment across cycles.
  • Implemented cycle based control logic with Rz axis quaternion interpolation and offset mapping, ensuring smooth transitions and accurate orientation.

Results

  • Achieved sub centimeter positional accuracy and stable orientation control across multiple pick and place cycles.
  • Reduced pose switching lag by 40 percent, improving real time response and reliability.
  • Delivered a modular vision integrated cobot cell deployable across multiple workflows.
